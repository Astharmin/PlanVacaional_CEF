NUESTROS PROGRAMAS

01 Programación para Niños

Introducción a la programación a través de juegos y actividades interactivas. Ideal para niños de 8 a 12 años.

Duración: 4 semanas

02 Robótica Educativa

Construye y programa tus propios robots. Talleres prácticos para todas las edades.

Duración: 5 semanas

03 Desarrollo Web

Aprende HTML, CSS y JavaScript para crear tus propias páginas web. Para adolescentes y adultos.

Duración: 6 semanas
